C++学习笔记

C++学习笔记

  • 第一位是19为十进制；第一位是0，第二位是17为八进制；第一位是0，第二位是X为十六进制
  • c++对字符使用单引号，对字符串使用双引号
  • sizeof运算符返回类型或变量的长度，单位为字节
  • 存储字符串的数组长度不能小于strlen(strings)+1
  • cin.getline(string, num): string数组，num读取的长度，最多读num-1个字符，遇到换行符停止读取
  • string对象在std命名空间下，和字符数组的主要区别是，可以将string对象声明为简单变量，而不是数组，string可以自动调整长度。可以将char数组视为用于存储一个字符串的char存储单元，而string类变量是一个表示字符串的实体
  • 不能将一个数组赋给另一个数组；可以将一个string赋给另一个string
  • string可以像数字一样加起来“aa” + “bb” = “aabb”； char数组用strcpy(str1, str2)赋值，用strcat(str1, str2)拼接
  • cout << R”(str)”，标识原始字符串，如果要表示 )”，可以用R”+* str)” +*”
  • 结构体初始化的时候各个值用逗号分开，不是用分号
  • new分配的内存都在堆(heap)，常规变量声明的内存在栈(stack)
  • int *p = new int[10] → delete [] p
  • C++管理内存的方式：自动存储(栈)，静态存储，动态存储(堆)、线程存储
  • vector使用的是堆，是动态数组的替代品； array使用的是栈，是静态数组的替代品；数组使用的是栈
  • 前缀版本(++x)比后缀(x++)版本效率高
  • strcmp比较字符串char不能用于string，如果是0则字符串相同
  • for和while循环都是入口条件循环，do while是出口条件循环
  • 逻辑符AND(&&)的优先级高于OR(||)，如果AND表达式的左侧为false则不会判断右侧的表达式
  • int * ar等价于int arr[]，arr[i] == *(ar + 1) &arr[i] == ar + 1
  • 禁止将const的地址赋给非const指针，可以使用const_cast强制类型转换来赋值
  • 只有一层间接关系时，才可以将非const地址或指针赋给const指针
  • int* const p1 = &num：可以使用p1改变num的值，但是p1只能指向num；
    const int* p2 = &num：p2可以指向别的数据，但是不能利用p2改变num的值
  • 结构名只是结构的名称，要获得结构的地址，必须使用地址运算符&
  • 构造函数：用函数
  • 构造函数不可以是虚函数，析构函数常常是虚函数：
    1.类中声明虚函数时，编译器会在类中生成一个虚函数表，虚函数表是一个存储成员函数指针的数据结构，虚函数表是由编译器自动生成和维护的，virtual成员函数会被编译器放入虚函数表中，当存在虚函数时，每个对象都有一个指向虚函数的指针。在实现多态的过程中，父类和派生类都有指向虚函数的指针。当对象在创建时，由编译器对指向虚函数的指针进行初始化，在定义子类对象时，指向虚函数的指针先指向父类的虚函数表，在父类构造完成以后，子类的指向虚函数的指针才能指向自己的虚函数表，如果构造函数是虚函数，那么调用构造函数就需要去找指向虚函数的指针，而此时指向虚函数的指针还没有初始化，因此，构造函数不可以是虚函数。
    2.析构函数可以声明为虚函数，类有继承时，析构函数常常必须是虚函数。若析构函数是虚函数，delete时基类和子类都会被释放，若析构函数不是虚函数，delete时只有基类会被释放，子类不会被释放，存在内存泄漏的隐患
  • 内联函数不能递归
  • 必须在声明引用变量时进行初始化
  • 可以通过初始化声明来设置引用，但不能通过赋值来设置
  • 按引用传递：引用被用作函数参数，使得函数中的变量名成为调用程序中的变量的别名
  • 传递引用的限制更严格，只能传递变量
  • 临时变量：如果实参与引用参数不匹配，将生成临时变量，如果引用参数是const，则在以下两种情况下生成临时变量：
  1. 实参类型正确，但不是左值
  2. 实参类型不正确，但可以转换为正确的类型
  • 使用const可以避免无意修改数据
    使用const可以处理const和非const实参，否则只能接受非const数据
    使用const引用使得函数能够正确生成并使用临时变量
  • 返回引用效率更高P267
  • 使用引用参数原因：1.修改引用的值； 2. 提高运行速度
  • 何时使用引用参数P274
  • 要为某个参数设置默认值，必须为它所有参数提供默认值：
    (int a = 1; int b = 2)可以
    (int a = 1; int b)不行
  • 函数重载时把类型引用和类型本身视为同一个特征标
