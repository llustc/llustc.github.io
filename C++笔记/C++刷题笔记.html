C++刷题笔记

刷题笔记

  • Int *(*ptr )()表示ptr是指向函数的指针，该函数的返回值是指向int型数据的指针
  • C++中不能建立引用数组和指向引用的指针，也不能建立引用的引用
  • 用Struct代替class的时候成员默认是public的
  • 关于类的大小的说明(待补充)：
    1.类的大小为类的非静态成员数据的类型大小之和，也就是说静态成员数据不作考虑。
    2.普通成员函数与sizeof无关。
    3.虚函数由于要维护在虚函数表，所以要占据一个指针大小，也就是4字节。
    4.类的总大小也遵守类似class字节对齐的，调整规则。
  • 数组名是一地址常量，数组名之间是不能彼此赋值的
  • 友元运算符被C++编译器的解释对应：
    a+b—>operator+(a, b); a.operator+(b)
    a++—>operator++(a, 0) a.operator++(0)
  • 引用是给对象取了一个别名，他引入了对象的同义词
  • 友元函数是普通函数，不能通过本类的this指针访问；友元函数是另一个类的成员函数，不能通过本类的this指针访问，只能通过另一个类的this指针访问。在C++中友元函数是独立于当前类的外部函数，一个友元函数可以同时定义为两个类的友元函数，友元函数既可以在类的内部，也可以在类的外部定义；在外部定义友元函数时，不必加关键字friend
  • 一个抽象类并不需要其中所有的方法都是抽象的
  • 组合和继承都可以的情况下，优先考虑组合
  • 继承的特点：
    1.公有继承的特点是基类的公有成员和保护成员作为派生类的成员时，它们都保持原有的状态，
    而基类的私有成员仍然是私有的，不能被这个派生类的子类所访问。
    2.私有继承的特点是基类的公有成员和保护成员都作为派生类的私有成员，并且不能被这个派生 类的子类所访问。
    3.保护继承的特点是基类的所有公有成员和保护成员都成为派生类的保护成员，并且只能被它的派生类成员函数或友元访问，基类的私有成员仍然是私有的。
  • 编译器会默认生成析构函数不是必须定义的 没有参数也没有返回值
  • 模板的使用是为了实现多态性
  • 运算符优先顺序：赋值运算符<逻辑运算符<关系运算符<算术运算符
  • struct成员类型不可以是它自己
  • 四种cast(C的强制转换不能进行错误检查)：
    • const_cast：const转为非const
    • static_const：非const转换为const、向上转换（子类到基类）
    • dynamic_const：用于动态类型转换，只能用于含有虚函数的类，只能转指针和引用，可以向上也可以向下
    • renterpret_cast：什么都可以转，可能会出现问题
  • 指针和引用的区别：
    • 指针有自己的空间，引用只是个别名
    • sizeof指针为4， sizeof引用为引用对象的大小
    • 指针用的时候需要解引用，引用可以直接改变引用对象的值
    • 指针可以被初始化为NULL，引用必须初始化为已有的对象
    • 可以有const指针，不能有const引用
    • 指针使用过程中可以指向其他变量，引用不能改变
    • 有多级指针，没有多级引用，引用只有一级
    • 指针++是地址自增，引用++是对象自增
    • 如果返回动态内存分配的对象或者内存，必须使用指针，使用引用可能会导致内存泄漏
  • 四种智能指针的区别：
    • auto_ptr：存在潜在的内存崩溃问题
    • unique_ptr：独占式拥有，同一时间只能有一个智能指针可以指向该对象，避免资源泄漏，使用std::move可以把一个unique指针赋给另一个
    • shared_ptr：共享式拥有，多个智能指针可以指向相同对象，该对象和其相关的资源会在最后一个引用被销毁的时候释放
    • weak_ptr：不控制对象生命周期，指向一个shared_ptr管理的对象，该对象的内存管理是shared_ptr，weak_ptr是为了防止两个shared_ptr互相引用导致计数器不为0的情况，他的引用不会导致计数器增加，不能通过weak_ptr直接访问对象的方法
  • 数组和指针的区别：
    • 数组：保存数据、直接访问数据、通常用于固定数目且数据类型相同的元素、隐式的分配和删除、自身即为数据名
    • 指针：保存数据地址，间接访问数据，先获取指针指向的地址，再读取该地址的数据、常用于动态的数据结构、通过malloc和free分配释放内存、通常指向匿名数据，操作匿名函数
  • 野指针：指向一个已删除的对象或者未申请访问受限内存区域的指针
  • 析构函数必须是虚函数，c++默认的析构函数不是虚函数：
    • 将可能会被继承的父类的析构函数设置成虚函数，可以保证我们new一个子类，然后使用基类指针指向该子类对象，释放基类指针时可以释放掉子类的空间，防止内存泄漏
    • 因为虚函数需要虚函数表和虚表指针，占用额外的内存，对于不会被继承的类来说，析构函数是虚函数就会浪费内存，因此C++默认的析构函数不是虚函数，只有需要当作父类的时候，设置为虚函数
  • 函数指针：函数指针是一个指针变量，指向一个具体的函数，可用该指针变量调用函数
  • fork函数？
  • C++析构函数的作用：
    • 析构函数和构造函数对应，命名相同，前面加一个~，对象所在的函数执行完毕时系统会自动执行析构函数
    • 只有一个析构函数，没有参数也没有返回值，不能重载
    • 如果没写析构函数，系统会自动生成一个缺省的析构函数，写了系统会合成一个析构函数，先调用写的，再调用合成的
    • 类中的指针需要先释放再调用析构函数
    • 类析构的顺序：派生类的析构函数→对象成员析构函数→基类的析构函数
  • 静态函数和虚函数的区别：
    • 静态函数在编译的时候确定运行顺序
    • 虚函数在运行的时候动态绑定，调用的时候会增加一次内存开销，因为用了虚函数表机制
  • static关键字：
    • 加了static关键字的全局变量只能在本文件中使用
    • static定义的静态局部变量分配在数据段上，普通的局部变量在栈上
    • 类中的成员变量和成员函数加了static就没有this指针了，只能通过类名访问
  • strcpy和strlen的区别：
    • strcpy拷贝到‘\0’结束，没有指定长度可能会越界，安全版本strncpy
    • strlen计算到‘\0’的长度

  • 虚函数和多态：
    • 多态主要是静态多态和动态多态，静态多态主要是重载，在编译时确定，动态多态是用虚函数机制实现的，在运行时动态绑定
    • 虚函数：在有虚函数的类中，类的最开始部分是一个虚函数表的指针，表中有虚函数的地址，子类继承父类中的虚函数会将其继承到的虚函数表中的地址替换为重写的函数地址，使用虚函数会增加访问内存开销，降低效率
  • 同时定义两个函数，一个带const，一个不带，这相当于函数重载
  • c++函数栈空间的最大值为1M
  • c++调用c函数需要extern “C”，因为C语言没有函数重载
  • new/delete：C++关键字，调用构造函数，返回的指针不用强转
    malloc/free：C语言函数库，必须指明申请的内存空间的大小，对于类类型的对象，不会调用构造函数和析构函数，返回的指针需要强转
  • RTTI(Run Time Type Identification)即通过运行时类型识别，程序能够使用基类的指针或引用来检查着这些指针或引用所指的对象的实际派生类型：
    • typeid操作符，返回指针和引用所指的实际类型；
    • dynamic_cast操作符，将基类类型的指针或引用安全地转换为其派生类类型的指针或引用。
  • C++拷贝赋值函数的形参能否进行值传递：不能，如果是这种情况下，调用拷贝构造函数的时候，首先要将实参传递给形参，这个传递的时候又要调用拷贝构造函数，如此循环，无法完成拷贝，栈也会满。
  • 继承访问权限的关系，无论成员被声明成什么类型，在类的外部，只能通过对象访问成员，并且通过对象只能访问public成员：
    • public公有继承：基类的所有公有成员都会成为派生类的公有成员，受保护成员成为派生类的受保护成员
    • private私有继承：基类的所有公有成员都会成为派生类的私有成员，受保护成员成为派生类的私有成员
    • protected受保护继承：基类的所有公有成员都会成为派生类的私有成员，受保护成员成为派生类的受保护成员
  • struct和class的区别：struct的默认继承权限和默认访问权限是public，class的默认继承权限和默认访问权限是private，class还可以定义模板类形参
  • 左值和右值的概念：
    • 左值：能对表达式取地址，或具体的对象和变量，一般指表达式结束后依然存在的持久对象
    • 右值：不能对表达式取地址，或者是匿名对象，一般指表达式结束就不再存在的临时对象
  • 左值引用和右值引用的区别：
    • 左值可以寻址，右值不可以
    • 左值可以被赋值，右值不可以被赋值，可以用来给左值赋值
    • 左值可变，右值不可变，用户自定义类型右值引用可以通过成员函数改变
  • 在32位机器上的C++语言中,int类型数据占4个字节,则short类型数据占 2个字节
  • 子类在内存中占用的字节数 = 父类占用字节数 + 自身成员所占的字节数，如果有虚函数，要算上虚函数表指针。
